# WhiteSur Panel Thin

Esta é uma configuração "magra" do painel do XFCE, sem referência ao plugin Windowck. Note que o plugin Appmenu ainda é necessário nesta versão.

O uso desta configuração é indicado em um ecossistema de aplicativos com decoração de janela no lado cliente (CSD), contexto em que o plugin Windowck torna-se uma duplicata não agradável dos controles de janela.
