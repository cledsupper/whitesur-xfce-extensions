## Variante escura
![Prévia da variante escura](https://i.imgur.com/XhTxXoh.png "Área de trabalho do XFCE com o Terminal do XFCE aberto com o Neofetch")

Talvez você queira os botões de janela similares ao Unity. Neste caso, recomendo este fork da extensão xfce4-windowck-plugin:
https://github.com/cledsupper/xfce4-windowck-plugin

A diferença do original, é que ele esconde os botões quando a janela maximizada não está em foco.

OBSERVAÇÃO: as imagens são ilustrativas. Os scripts deste repositório não fazem o milagre dos borrões nos fundos do painel e das janelas aparecer. Para isso, procure o compositor "picom".
